
# DeadalusPHP



## Run the computer

Clone the project

```bash
  git clone https://github.com/DeadalusPHP/freamwork.git
```

Go to the project directory

```bash
  cd freamwork
```

Run with docker

```bash
  docker compose up -d --build
  docker exec -it DeadalusPHP-web-1 bash
```

Load the dependencies

```bash
  composer install
```
